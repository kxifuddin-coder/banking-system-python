# Banking System - Python OOP Project

A comprehensive banking system built from scratch using Python OOP principles.

## Features
- Multiple account types (Regular, Savings, Checking)
- Deposit & Withdrawal operations
- Interest calculations for Savings accounts
- Overdraft protection for Checking accounts
- Account comparison and merging
- Transaction history tracking
- Data persistence using pickle

## Installation

1. Clone this repository:
```bash
git clone https://github.com/kxifuddin-coder/banking-system-python.git
```

2. Navigate to the project directory:
```bash
cd banking-system-python
```

3. Run the program:
```bash
python class_Bnking_system.py
```

## Requirements
- Python 3.10+ (uses match-case statement)
- No external libraries required (uses only built-in modules)

## How to Use
1. Run the program
2. Choose from the menu options (1-10)
3. Follow the on-screen prompts
4. Data is automatically saved to `bank_data.pkl`

## Learning Focus
This project demonstrates:
- Class inheritance
- Magic methods (`__init__`, `__str__`, `__add__`, `__eq__`, `__gt__`, `__lt__`)
- Encapsulation
- Polymorphism
- File handling with pickle

This is a learning project built to master Python OOP concepts. All core logic written manually.
